What I did in the Signy Adventuryy

Our team of 5 students built a custom metroidvania game

Technology

I used Unity with c# to reimplement the game, behavior tree to implement boss fight

My contribution

I am the leader of the game project group. I created tasks every week and distribute them to group members.

I designed the whole game, including every part of our game. This includes all mechanisms, player mechanism, all enemies design, entire map design and built the map, boss design, UI design, and designed for what system we need to implement.

I used queue and linked list to implement an Audio Source Pool to instantiate audio source at any position using object pool mechanism. And I implemented the entire player with good feeling, including move, jump, double jump, block, block jump, multiple types of attack, 5 abilities power up, interaction with enemy, invincibility, screen shake effect, audio, and entire animation system.

I also implemented the boss fight, including all animation and boss mechanism using behavior tree.
